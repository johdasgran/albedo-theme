## :wrench: Installation
### :one: <b>First installation way</b>

- Open terminal.

- Clone this repository

    ```
    git clone https://github.com/johdasgran/jinx-theme.git
    ```

- Change directory to the Jinx Theme

    ```
    cd jinx-theme
    ```

 - Setup the new GRUB

    ```
    sudo ./install.sh
    sudo update-grub
    ```
 - Reboot your PC
